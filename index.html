JSLightBoCSS for Egg Maps.

JSLightBoCSS is an almost-pure™ CSS lightbox that has only a bit of JS (to provide lazy-loading of large images).
JSLightBoCSS is based on LightBoCSS, which aggressively loads all images referenced.

JSLightBoCSS loads large images only as-needed. This will save you and your users bandwidth if they don't look at all of your images. The down-side is that your users will have to wait for the large image to load.

Feel free to use the code, according to the terms of the license.

JSLightBoCSS was created for Egg Maps (a Google Maps app for iOS6. Check out Egg Maps: http://www.eggmaps.com/ ).

You can browse the HTML, the CSS, download a zip file, or browse the GitHub repository.
